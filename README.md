# 🚀 AI Powered Study Planner

A modern productivity web application that helps students organize their study tasks, track progress, and stay focused using a Pomodoro timer.

This project was built using AI tools and deployed on Vercel as part of an internship assignment.

---

# 🌐 Live Demo

https://study-planner-beige-eight.vercel.app/

---

# 📌 Features

### 🔐 User Authentication
- Register and Login system
- Personalized dashboard
- User data stored securely using localStorage

### 📚 Task Management
- Add study tasks
- Set subject and priority
- Set deadlines
- Mark tasks as completed
- Delete tasks
- Attach study materials (PDF / images)

### 📊 Productivity Dashboard
- Total tasks
- Completed tasks
- Pending tasks
- Visual progress bar

### ⏱ Pomodoro Focus Timer
- 25 minute focus session
- 5 minute break session
- Session counter
- Start / Pause / Reset

### 🤖 Smart AI Suggestions
The system analyzes deadlines and priorities to suggest which task should be completed next.

### 🔔 Notifications
Deadline reminders are sent using browser notifications.

### 📁 Export Options
Users can export their study plan as:
- CSV file
- PDF report

### 🎨 UI Features
- Modern glassmorphism UI
- Dark mode and light mode support
- Clean responsive layout

# 🎯 Problem It Solves

Students often struggle with managing multiple assignments and maintaining focus during study sessions.

This application provides a single platform to:
- Organize study tasks
- Track progress
- Improve focus
- Manage deadlines efficiently
